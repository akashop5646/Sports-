# CreaseLive - Backend Service

This is a placeholder for the future backend service of the CreaseLive cricket tournament scoring and analytics app.

## Proposed Tech Stack
- **Runtime**: Node.js / Bun
- **Framework**: Express, Elysia, or Hono
- **Database**: PostgreSQL or MongoDB (with Prisma or Mongoose)
- **Real-time**: WebSockets for live ball-by-ball score updates
